# Visitor & Employee Gate Pass Management System
SecurePass 360 is a production-ready ASP.NET Core MVC application built to streamline visitor management, employee attendance, and gate security using QR-based validation.

The system provides a centralized admin control panel for managing all categories of employees — including office staff, security personnel, contract workers, and management — while ensuring secure, real-time monitoring of entry and exit activities.

Designed for corporate and institutional environments, SecurePass 360 emphasizes:

🔒 Full-cycle security with QR validation

📊 Real-time analytics for attendance and visitor tracking

🛠️ Scalable architecture using ASP.NET Core + SQL Server (ADO.NET)

🎨 Professional UI/UX with ID card–style layouts and responsive design
